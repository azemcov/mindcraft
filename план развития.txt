1. Пользователи
	- пользователь id
	- пользователь уникальный ник
	- mail (обязательно ли?)
	- пользователь имя (если он хочет Tomas Muller например)
	- пользователь должность
	- хэш-пароль
	- соль
	- дата регистрации
	- аватар-картинка
	- в каких группах состоит как создатель [id группы]
	- в каких группах состоит как сдающий [id группы]
	- создалеть тестов [id тестов]
	- пройденные тесты (нужны ли?)
	- ответы на пройденый тесты (нужны ли?)
	- (еще какие-то данные)

2. Группы
	- id группы
	- Название группы
	- Владелец
	- дата создания (надо ли?)
	- Участники (сдающие) []
	- Открытая/закрытая (по согласованию с владельцем либо по приглашению)
	- Тесты группы [id теста]

3. Тесты
	- id теста
	- Название теста
	- Владелец (создатель)
	- Автопроверяемый или ручная проверка создателем после сдачи
	- Дата создания
	- Список вопросов [id вопроса]
	- Дата с какого момента доступен
	- Дата до какого момента доступен
	- Время отведённое на выполнение
	- Максимальный балл
	
4. enum Шаблоны (по сути вопрос может быть текстовый, текст+картинка, текст+аудио и пр. и ответ также)
	- TEXT
	- PIC
	- AUDIO
	
5. Вопросы
	- id вопроса
	- шаблон вопроса
	- шаблон ответа
	- текст вопроса
	- текст ответа [варианты]
	- картинка вопроса
	- картинка ответа [варианты]
	- аудио вопроса
	- аудио ответа [варианты]
	- верный вариант индекс в массиве (если создатель выбрал автопроверяемый)
